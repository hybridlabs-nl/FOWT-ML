# FOWT-ML

Machine learning kit for Floating Offshore Wind Turbines (FOWT).

## Installation

To install `FOWT-ML` from GitHub repository, do:

```console
git clone git@github.com:hybridlabs-nl/fowt-ml.git
cd FOWT-ML
python -m pip install .
```

To install the package in development mode, you can clone the repository and
install it using pip:

```bash
pip install -e .[dev]
```

To work with notebooks, you need to install `jupyterlab`:

```bash
pip install jupyterlab
```
